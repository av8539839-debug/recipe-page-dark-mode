# recipe-page-dark-mode
 Receitas Dinâmicas com HTML, CSS e JS  Neste projeto, explorei a manipulação do DOM e a estilização avançada com CSS. Além da estrutura semântica, implementei um sistema de troca de temas (Light/Dark Mode) e utilizei o objeto Date() do JavaScript para criar uma interação personalizada de boas-vindas. Um passo importante para a minha jornada
