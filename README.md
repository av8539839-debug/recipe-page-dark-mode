# 🥞 Página de Receitas Interativa

Este é um projeto de uma página de receitas (Panquecas Clássicas) desenvolvido para praticar os fundamentos do desenvolvimento Web Front-end. A página conta com interatividade via JavaScript, incluindo saudações dinâmicas e alternância de temas.

## 🚀 Funcionalidades

* **Dark Mode:** Alternância entre tema claro e escuro para melhor conforto visual.
* **Saudação Dinâmica:** O site detecta o horário do usuário e exibe "Bom dia", "Boa tarde" ou "Boa noite" via alerta.
* **Feedback de Conclusão:** Botão interativo que sinaliza quando a receita foi finalizada.
* **Responsividade:** Layout adaptável para dispositivos móveis e desktop.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização com Flexbox, variáveis e transições suaves.
* **JavaScript (Vanilla):** Lógica para manipulação do DOM e controle de temas.
